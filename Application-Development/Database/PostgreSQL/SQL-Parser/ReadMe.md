# PostgreSQL Parser

## pgsanity (Python)

`pgsanity` is a Python library that provides a simple interface for parsing and analyzing PostgreSQL SQL queries. It can be used to extract information about the structure of SQL queries, such as the tables and columns involved, as well as to perform various checks and transformations on the queries.

### Installation

```bash
pip install pgsanity
```

### Usage (CLI)

```bash
pgsanity some_query.sql
```
