# Security Relevante GitHub Repositories

If you’re starting your cybersecurity journey for FREE, these GitHub repos will save you months of confusion and give you real-world hands-on experience.

## WebGoat (OWASP)
Practice real OWASP Top 10 vulnerabilities with guided labs
<https://github.com/WebGoat/WebGoat>

## OWASP Juice Shop
One of the best intentionally vulnerable web applications
Perfect for XSS, SQLi, IDOR & logic flaws
<https://github.com/juice-shop/juice-shop>

## DVWA (Damn Vulnerable Web App)
A classic beginner-friendly vulnerable app with adjustable difficulty
<https://github.com/digininja/DVWA>

## HackTricks
Your all-in-one cheat sheet for payloads, enumeration & attack logic
<https://github.com/HackTricks-wiki/hacktricks>

## PayloadsAllTheThings
Real-world payload examples for XSS, SQLi, SSRF, command injection & more
<https://github.com/swisskyrepo/PayloadsAllTheThings>

## The Art of Hacking
Massive curated collection of free labs, books, tools & resources
<https://github.com/The-Art-of-Hacking/h4cker>

## Awesome Bug Bounty
Everything you need to start bug bounty
<https://github.com/djadmin/awesome-bug-bounty>
